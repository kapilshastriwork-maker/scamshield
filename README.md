## Problem Statement

Online scams are becoming increasingly sophisticated, affecting millions of people worldwide. From fake prize notifications to banking phishing attempts, scammers use various tactics to deceive victims. There's a critical need for a tool that can instantly analyze suspicious messages and provide actionable insights to help users stay safe.

## Solution

ScamShield is an AI-powered scam detection application that uses Google Gemini to analyze suspicious messages in real-time. Users can paste any message‚Äîwhether it's an SMS, email, or chat‚Äîand receive an instant verdict with confidence score, detailed reasons, and personalized advice.

## How to Run Locally

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd dsoc-hack
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Create a `.env` file in the `src` folder with your Gemini API key:
   ```
   VITE_GEMINI_API_KEY=your_api_key_here
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and navigate to `http://localhost:5173`
